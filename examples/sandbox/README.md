# Example Sandbox

This directory is a minimal in-repo example of the target structure OpenTop expects in a working repository.

It is not meant to replace your real `OpenTop-Sandbox` repo. It exists so new contributors can inspect a representative `.opentop/` layout without hunting through external local directories.

Contents:

- `.opentop/opentop.yml`
- `.opentop/project-context.md`
- `.opentop/rules.md`
